# Flask CRUD Tasks (SQLite)

A simple full CRUD app:
- Create tasks
- Read/list tasks (with search)
- Update tasks (edit + toggle done)
- Delete tasks

## Tech
- Python (Flask)
- SQLite
- Vanilla JS (fetch API)
- HTML/CSS

## Run locally
```bash
python -m venv .venv
# Windows:
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
